Security Restrictions

QuickPay enforces the following security measures at the application level:

  • Seed Vault only: Wallet authorization is restricted exclusively to the Solana Mobile Seed Vault. Connections from third-party wallet apps (e.g., Phantom, Solflare) are blocked. This is a deliberate design decision to prevent unauthorized transaction signing.
  • Token whitelist: Incoming SPL token payments are validated against a hardcoded whitelist of approved mint addresses (SOL, USDC, USDT, SKR). Payments from unrecognized or mismatched token programs are automatically rejected to protect users from scam tokens.

These restrictions are non-configurable and exist solely for user protection.

Storage Permissions

QuickPay requests storage write permission for the following purposes:

  • Receipt images: Saving PNG receipt images to Pictures/QuickPay/ on your device when you tap "Save to Gallery".
  • History backups: Saving JSON backup files to Downloads/QuickPay/ when you tap the backup button in the History screen. These files can be restored into the app at any time.

Storage is accessed only upon explicit user action. No files are written automatically or in the background.
